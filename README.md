# UsoIA_Gomez_Serrano_Sebastian / Manipulación API Rick and Morty

## Introducción

En este repositorio se utiliza la API de Rick y Morty para hacer una solicitud al link de la API y obtener los personajes deseados filtrandolos mediante un buscador que funciona encontrando los personajes en base a su nombre, funcionando tambien con solo letras clave.
